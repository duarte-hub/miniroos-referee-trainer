# MiniRoos Referee Trainer

A Home Assistant add-on repository containing an ingress web app that helps referees build confidence and learn the MiniRoos playing formats and rules through multiple-choice scenarios.

## What it does

- Organises rule-learning by `Under 6 & 7`, `Under 8 & 9`, and `Under 10 & 11`
- Uses multiple-choice questions backed by the supplied Football Australia MiniRoos rules PDF
- Shows immediate explanations after each answer to reinforce learning
- Tracks score and confidence-building progress inside the browser

## Install in Home Assistant

1. Push this repository to GitHub.
2. In Home Assistant, open `Settings -> Add-ons -> Add-on Store -> Repositories`.
3. Add your GitHub repository URL.
4. Install `MiniRoos Referee Trainer`.
5. Open the add-on from the sidebar or via ingress.
